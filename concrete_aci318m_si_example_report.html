Concrete Beam Analysis - ACI 318M-25 (SI Units)

Determination of neutral axis depth for singly reinforced concrete beam using metric units.

Code References

[ASSUME] ACI 318M-25 Building Code Requirements for Structural Concrete (Metric)

[ASSUME] SI units (Systeme International) used throughout

[ASSUME] Normal weight concrete assumed

1. Material Properties

Specified compressive strength of concrete;

\( f_{c}^{\prime} = 30 \ \mathrm{MPa} \)

[ACI 318M-25 19.2.1.1]

Specified yield strength of reinforcement;

\( f_y = 420 \ \mathrm{MPa} \)

[ACI 318M-25 20.2.2.1]

Modulus of elasticity of steel

\( E_s = 200000 \ \mathrm{MPa} \)

[ACI 318M-25 20.2.2.2]

2. Geometric Properties

Width of compression face of beam;

\( b = 300 \ \mathrm{mm} \)

Overall depth of beam;

\( h = 600 \ \mathrm{mm} \)

Concrete cover to reinforcement center;

\( cover = 40 \ \mathrm{mm} \)

[ACI 318M-25 Table 20.5.1.3.1]

Effective depth to tension reinforcement

\[ \begin{align} d &= {h} - {cover} = 600 \ \mathrm{mm} - 40 \ \mathrm{mm} \\[4pt] & \therefore d = 560 \ \mathrm{mm} \end{align} \]

Area of tension reinforcement;

\( A_s = 2000 \ \mathrm{mm^2} \)

3. Stress Block Parameters

Stress block factor

\( \beta_1 = 0.85 \ \mathrm{} \)

[ACI 318M-25 Table 22.2.2.4.3]

4. Neutral Axis Calculation

Depth of equivalent rectangular stress block

\[ \begin{align} a &= \frac{ {A_s} \cdot {f_y} }{ {0.85} \cdot {f_{c}^{\prime}} \cdot {b} } \\[4pt] &= \frac{ 2000 \ \mathrm{mm^2} \cdot 420 \ \mathrm{MPa} }{ 0.85 \ \mathrm{} \cdot 30 \ \mathrm{MPa} \cdot 300 \ \mathrm{mm} } \\[4pt] &\therefore a = 109.8 \ \mathrm{mm} \end{align} \]

Distance from extreme compression fiber to neutral axis

\[ \begin{align} c &= \frac{ {a} }{ {\beta_1} } = \frac{ 109.8 \ \mathrm{mm} }{ 0.85 \ \mathrm{} } \\[4pt] & \therefore c = 129.2 \ \mathrm{mm} \end{align} \]

[ACI 318M-25 22.2.2.4.1]

5. Strain Compatibility Check

Maximum usable strain in concrete

\( \epsilon_{cu} = 0.003 \ \mathrm{} \)

[ACI 318M-25 22.2.2.1]

Strain in tension reinforcement

\[ \begin{align} \epsilon_s &= \frac{ {\epsilon_{cu}} \cdot {d} - {c} }{ {c} } = \frac{ 0.003 \ \mathrm{} \cdot 560 \ \mathrm{mm} - 129.2 \ \mathrm{mm} }{ 129.2 \ \mathrm{mm} } \\[4pt] & \therefore \epsilon_s = 0.01001 \ \mathrm{} \end{align} \]

Yield strain of reinforcement

\[ \begin{align} \epsilon_y &= \frac{ {f_y} }{ {E_s} } = \frac{ 420 \ \mathrm{MPa} }{ 200000 \ \mathrm{MPa} } \\[4pt] & \therefore \epsilon_y = 0.0021 \ \mathrm{} \end{align} \]

6. Section Classification

Steel yield check

\[ \begin{align} Check \ \epsilon_s \ & \geq \ \epsilon_y \\[4pt] \ 0.01001 \ \mathrm{} \ & \geq \ 0.0021 \ \mathrm{} \\[4pt] &\therefore Tension-controlled section (phi = 0.9) \end{align} \]

[ACI 318M-25 21.2.2]

Minimum required reinforcement

\[ \begin{align} A_{s,min} &= \max{\left( \frac{ {0.25} \cdot \sqrt{ {f_{c}^{\prime}} } \cdot {b} \cdot {d} }{ {f_y} }, \frac{ {1.4} \cdot {b} \cdot {d} }{ {f_y} } \right)} \\[4pt] &= \max{\left( \frac{ 0.25 \ \mathrm{} \cdot \sqrt{ 30 \ \mathrm{MPa} } \cdot 300 \ \mathrm{mm} \cdot 560 \ \mathrm{mm} }{ 420 \ \mathrm{MPa} }, \frac{ 1.4 \ \mathrm{} \cdot 300 \ \mathrm{mm} \cdot 560 \ \mathrm{mm} }{ 420 \ \mathrm{MPa} } \right)} \\[4pt] &\therefore A_{s,min} = 560 \ \mathrm{mm^2} \end{align} \]

[ACI 318M-25 9.6.1.2]

Minimum reinforcement check

\[ \begin{align} Check \ A_s \ & \geq \ A_{s,min} \\[4pt] \ 2000 \ \mathrm{mm^2} \ & \geq \ 560 \ \mathrm{mm^2} \\[4pt] &\therefore OK - Minimum reinforcement satisfied \end{align} \]

7. Nominal Moment Capacity

Nominal flexural strength

\[ \begin{align} M_n &= \frac{ {A_s} \cdot {f_y} \cdot {d} - \frac{ {a} }{ {2} } }{ {1e+06} } \\[4pt] &= \frac{ 2000 \ \mathrm{mm^2} \cdot 420 \ \mathrm{MPa} \cdot 560 \ \mathrm{mm} - \frac{ 109.8 \ \mathrm{mm} }{ 2 \ \mathrm{} } }{ 1000000 \ \mathrm{} } \\[4pt] &\therefore M_n = 424.3 \ \mathrm{kN*m} \end{align} \]

[ACI 318M-25 22.2.2.4.1]

Strength reduction factor for flexure

\( \phi = 0.9 \ \mathrm{} \)

[ACI 318M-25 21.2.1]

Design flexural strength

\[ \begin{align} M_d &= {\phi} \cdot {M_n} = 0.9 \ \mathrm{} \cdot 424.3 \ \mathrm{kN*m} \\[4pt] & \therefore M_d = 381.9 \ \mathrm{kN*m} \end{align} \]

8. Summary

This analysis demonstrates ACI 318M-25 compliance using SI units. The neutral axis depth and moment capacity have been calculated according to metric building code requirements.
