Rectangular HSS Compression Design (SI Units)

[ASSUME] Members are in pure compression

[ASSUME] AISC principles adapted for metric units

[ASSUME] Torsional unbraced length does not exceed lateral unbraced length

1. Inputs

Ultimate compressive load;

\( P_u = 45 \ \mathrm{kN} \)

Member length;

\( L = 1.2 \ \mathrm{m} \)

Member effective length factor;

\( k_{c} = 1 \ \mathrm{} \)

Section width;

\( b = 152 \ \mathrm{mm} \)

Section height;

\( h = 51 \ \mathrm{mm} \)

Wall thickness;

\( t = 3.2 \ \mathrm{mm} \)

Material yield stress;

\( F_{y} = 248 \ \mathrm{MPa} \)

Modulus of elasticity;

\( E = 200000 \ \mathrm{MPa} \)

Resistance factor for compression;

\( \phi_c = 0.9 \ \mathrm{} \)

2. Section Properties

Gross cross-sectional area

\[ \begin{align} A_{g} &= {b} \cdot {h} - {b} - {2} \cdot {t} \cdot {h} - {2} \cdot {t} \\[4pt] &= 152 \ \mathrm{mm} \cdot 51 \ \mathrm{mm} - 152 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \cdot 51 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \\[4pt] &\therefore A_{g} = 1258 \ \mathrm{mm^2} \end{align} \]

Moment of inertia about x-axis

\[ \begin{align} I_x &= \frac{ {b} \cdot {\left( {h} \right)}^{ {3} } - {b} - {2} \cdot {t} \cdot {\left( {h} - {2} \cdot {t} \right)}^{ {3} } }{ {12} } \\[4pt] &= \frac{ 152 \ \mathrm{mm} \cdot {\left( 51 \ \mathrm{mm} \right)}^{ 3 \ \mathrm{} } - 152 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \cdot {\left( 51 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \right)}^{ 3 \ \mathrm{} } }{ 12 \ \mathrm{} } \\[4pt] &\therefore I_x = 603819 \ \mathrm{mm^4} \end{align} \]

Moment of inertia about y-axis

\[ \begin{align} I_y &= \frac{ {h} \cdot {\left( {b} \right)}^{ {3} } - {h} - {2} \cdot {t} \cdot {\left( {b} - {2} \cdot {t} \right)}^{ {3} } }{ {12} } \\[4pt] &= \frac{ 51 \ \mathrm{mm} \cdot {\left( 152 \ \mathrm{mm} \right)}^{ 3 \ \mathrm{} } - 51 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \cdot {\left( 152 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} \right)}^{ 3 \ \mathrm{} } }{ 12 \ \mathrm{} } \\[4pt] &\therefore I_y = 3453221 \ \mathrm{mm^4} \end{align} \]

Radius of gyration about x-axis

\[ \begin{align} r_x &= \sqrt{ \frac{ {I_x} }{ {A_{g}} } } = \sqrt{ \frac{ 603819 \ \mathrm{mm^4} }{ 1258 \ \mathrm{mm^2} } } \\[4pt] & \therefore r_x = 21.91 \ \mathrm{mm} \end{align} \]

Radius of gyration about y-axis

\[ \begin{align} r_y &= \sqrt{ \frac{ {I_y} }{ {A_{g}} } } = \sqrt{ \frac{ 3453221 \ \mathrm{mm^4} }{ 1258 \ \mathrm{mm^2} } } \\[4pt] & \therefore r_y = 52.39 \ \mathrm{mm} \end{align} \]

Width-to-thickness ratio

\[ \begin{align} \mathrm{b/t} &= \frac{ {b} - {2} \cdot {t} }{ {2} \cdot {t} } = \frac{ 152 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} }{ 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} } \\[4pt] & \therefore \mathrm{b/t} = 22.75 \ \mathrm{} \end{align} \]

Height-to-thickness ratio

\[ \begin{align} \mathrm{h/t} &= \frac{ {h} - {2} \cdot {t} }{ {2} \cdot {t} } = \frac{ 51 \ \mathrm{mm} - 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} }{ 2 \ \mathrm{} \cdot 3.2 \ \mathrm{mm} } \\[4pt] & \therefore \mathrm{h/t} = 6.969 \ \mathrm{} \end{align} \]

3. Buckling Properties

Element compression slenderness limit

\[ \begin{align} \lambda_{r} &= {1.4} \cdot \sqrt{ \frac{ {E} }{ {F_{y}} } } = 1.4 \ \mathrm{} \cdot \sqrt{ \frac{ 200000 \ \mathrm{MPa} }{ 248 \ \mathrm{MPa} } } \\[4pt] & \therefore \lambda_{r} = 39.76 \ \mathrm{} \end{align} \]

[AISC Table B4.1a equivalent]

Maximum element slenderness ratio

\[ \begin{align} \lambda_{max} &= \max{\left( {\mathrm{b/t}}, {\mathrm{h/t}} \right)} = \max{\left( 22.75 \ \mathrm{}, 6.969 \ \mathrm{} \right)} \\[4pt] & \therefore \lambda_{max} = 22.75 \ \mathrm{} \end{align} \]

Member slenderness ratio

\[ \begin{align} \mathrm{KL/r} &= \frac{ {k_{c}} \cdot {L} \cdot {1000} }{ \min{\left( {r_x}, {r_y} \right)} } \\[4pt] &= \frac{ 1 \ \mathrm{} \cdot 1.2 \ \mathrm{m} \cdot 1000 \ \mathrm{} }{ \min{\left( 21.91 \ \mathrm{mm}, 52.39 \ \mathrm{mm} \right)} } \\[4pt] &\therefore \mathrm{KL/r} = 54.78 \ \mathrm{} \end{align} \]

4. Compressive Strength

Elastic buckling stress

\[ \begin{align} F_e &= \frac{ {\left( {\pi} \right)}^{ {2} } \cdot {E} }{ {\left( {\mathrm{KL/r}} \right)}^{ {2} } } \\[4pt] &= \frac{ {\left( 3.142 \ \mathrm{} \right)}^{ 2 \ \mathrm{} } \cdot 200000 \ \mathrm{MPa} }{ {\left( 54.78 \ \mathrm{} \right)}^{ 2 \ \mathrm{} } } \\[4pt] &\therefore F_e = 657.8 \ \mathrm{MPa} \end{align} \]

[AISC Eq. E3-4 equivalent]

\[ \begin{align} Check \ \lambda_{max} \ & < \ \lambda_{r} \\[4pt] \ 22.75 \ \mathrm{} \ & < \ 39.76 \ \mathrm{} \\[4pt] &\therefore Non-Slender \end{align} \]

\[ \begin{align} \lambda_{crit} &= {4.71} \cdot \sqrt{ \frac{ {E} }{ {F_{y}} } } = 4.71 \ \mathrm{} \cdot \sqrt{ \frac{ 200000 \ \mathrm{MPa} }{ 248 \ \mathrm{MPa} } } \\[4pt] & \therefore \lambda_{crit} = 133.8 \ \mathrm{} \end{align} \]

[AISC Section E3 equivalent]

\[ \rightarrow \mathrm{KL/r} \ \leq \ \lambda_{crit} \]

...Inelastic buckling controls.

Critical compressive stress

\[ \begin{align} F_{cr} &= {F_{y}} \cdot {\left( {0.658} \right)}^{ \frac{ {F_{y}} }{ {F_e} } } \\[4pt] &= 248 \ \mathrm{MPa} \cdot {\left( 0.658 \ \mathrm{} \right)}^{ \frac{ 248 \ \mathrm{MPa} }{ 657.8 \ \mathrm{MPa} } } \\[4pt] &\therefore F_{cr} = 211.8 \ \mathrm{MPa} \end{align} \]

[AISC Eq. E3-2 equivalent]

Nominal member compressive strength

\[ \begin{align} P_n &= \frac{ {F_{cr}} \cdot {A_{g}} }{ {1000} } = \frac{ 211.8 \ \mathrm{MPa} \cdot 1258 \ \mathrm{mm^2} }{ 1000 \ \mathrm{} } \\[4pt] & \therefore P_n = 266.5 \ \mathrm{kN} \end{align} \]

[AISC Eq. E3-1 equivalent]

5. Member Demand vs. Capacity Check

Design member compressive capacity

\[ \begin{align} \phi P_n &= {\phi_c} \cdot {P_n} = 0.9 \ \mathrm{} \cdot 266.5 \ \mathrm{kN} \\[4pt] & \therefore \phi P_n = 239.8 \ \mathrm{kN} \end{align} \]

\[ \begin{align} Check \ P_u \ & \leq \ \phi P_n \\[4pt] \ 45 \ \mathrm{kN} \ & \leq \ 239.8 \ \mathrm{kN} \\[4pt] &\therefore OK \end{align} \]
