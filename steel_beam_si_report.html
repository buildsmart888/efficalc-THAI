Steel Beam Moment Strength (SI Units)

Flexural design strength of a steel wide-flange beam section using metric units.

Assumptions

[ASSUME] AISC principles adapted for metric units

[ASSUME] Beam web is unstiffened

[ASSUME] Beam design is not controlled by deflection requirements

Inputs

Beam ultimate moment demand;

\( M_u = 40 \ \mathrm{kN*m} \)

Beam unbraced length;

\( L_b = 6 \ \mathrm{m} \)

Beam section (W18X40 equivalent);

\( \mathrm{section} = \mathrm{W460X60} \ \)

Steel yield strength;

\( F_y = 345 \ \mathrm{MPa} \)

Modulus of elasticity;

\( E = 200000 \ \mathrm{MPa} \)

Lateral-torsional buckling modification factor;

\( C_b = 1 \ \mathrm{} \)

[AISC F1(3) equivalent]

Section Properties

\[ section = {W460X60 (equivalent to W18X40)} \]

Section modulus

\( S_x = 1050000 \ \mathrm{mm^3} \)

Plastic section modulus

\( Z_x = 1170000 \ \mathrm{mm^3} \)

Radius of gyration about y-axis

\( r_{y} = 50.8 \ \mathrm{mm} \)

Effective radius of gyration

\( r_{ts} = 63.5 \ \mathrm{mm} \)

Torsional constant

\( J = 267000 \ \mathrm{mm^4} \)

Distance between flange centroids

\( h_o = 430 \ \mathrm{mm} \)

Flange local buckling parameter

\( b_f/2t_f = 7.5 \ \mathrm{} \)

Web local buckling parameter

\( h/t_w = 35 \ \mathrm{} \)

1. Beam Flexural Capacity

Flexural resistance factor

\( \phi_{b} = 0.9 \ \mathrm{} \)

[AISC F1(1) equivalent]

1.1. Section Compactness

\[ \begin{align} \lambda_{pf} &= {0.38} \cdot \sqrt{ \frac{ {E} }{ {F_y} } } = 0.38 \ \mathrm{} \cdot \sqrt{ \frac{ 200000 \ \mathrm{MPa} }{ 345 \ \mathrm{MPa} } } \\[4pt] & \therefore \lambda_{pf} = 9.149 \ \mathrm{} \end{align} \]

[AISC Table B4.1b(10) equivalent]

\[ \begin{align} Check \ b_f/2t_f \ & \leq \ \lambda_{pf} \\[4pt] \ 7.5 \ \mathrm{} \ & \leq \ 9.149 \ \mathrm{} \\[4pt] &\therefore CompactFlange \end{align} \]

\[ \begin{align} \lambda_{pw} &= {3.76} \cdot \sqrt{ \frac{ {E} }{ {F_y} } } = 3.76 \ \mathrm{} \cdot \sqrt{ \frac{ 200000 \ \mathrm{MPa} }{ 345 \ \mathrm{MPa} } } \\[4pt] & \therefore \lambda_{pw} = 90.53 \ \mathrm{} \end{align} \]

[AISC Table B4.1b(15) equivalent]

\[ \begin{align} Check \ h/t_w \ & \leq \ \lambda_{pw} \\[4pt] \ 35 \ \mathrm{} \ & \leq \ 90.53 \ \mathrm{} \\[4pt] &\therefore CompactWeb \end{align} \]

1.2. Plastic Moment Strength

Nominal plastic moment strength

\[ \begin{align} M_{p} &= \frac{ {F_y} \cdot {Z_x} }{ {1e+06} } = \frac{ 345 \ \mathrm{MPa} \cdot 1170000 \ \mathrm{mm^3} }{ 1000000 \ \mathrm{} } \\[4pt] & \therefore M_{p} = 403.6 \ \mathrm{kN*m} \end{align} \]

[AISC Eq. F2-1 equivalent]

1.3. Yielding Strength

\[ \begin{align} M_{ny} &= \frac{ {F_y} \cdot {Z_x} }{ {1e+06} } = \frac{ 345 \ \mathrm{MPa} \cdot 1170000 \ \mathrm{mm^3} }{ 1000000 \ \mathrm{} } \\[4pt] & \therefore M_{ny} = 403.6 \ \mathrm{kN*m} \end{align} \]

[AISC Eq. F2-1 equivalent]

1.4. Lateral-Torsional Buckling

\[ \begin{align} L_{p} &= \frac{ {1.76} \cdot {r_{y}} \cdot \sqrt{ \frac{ {E} }{ {F_y} } } }{ {1000} } \\[4pt] &= \frac{ 1.76 \ \mathrm{} \cdot 50.8 \ \mathrm{mm} \cdot \sqrt{ \frac{ 200000 \ \mathrm{MPa} }{ 345 \ \mathrm{MPa} } } }{ 1000 \ \mathrm{} } \\[4pt] &\therefore L_{p} = 2.153 \ \mathrm{m} \end{align} \]

[AISC Eq. F2-5 equivalent]

\( c = 1 \ \mathrm{} \)

[AISC Eq. F2-8a equivalent]

\[ \begin{align} L_{r} &= \frac{ \frac{ {1.95} \cdot {r_{ts}} }{ {1000} } \cdot {E} }{ {0.7} \cdot {F_y} } \cdot \sqrt{ \frac{ {J} \cdot {c} }{ {S_x} \cdot {h_o} } + \sqrt{ {\left( \frac{ {J} \cdot {c} }{ {S_x} \cdot {h_o} } \right)}^{ {2} } + {6.76} \cdot {\left( \frac{ {0.7} \cdot {F_y} }{ {E} } \right)}^{ {2} } } } \\[4pt] &= \frac{ \frac{ 1.95 \ \mathrm{} \cdot 63.5 \ \mathrm{mm} }{ 1000 \ \mathrm{} } \cdot 200000 \ \mathrm{MPa} }{ 0.7 \ \mathrm{} \cdot 345 \ \mathrm{MPa} } \cdot \sqrt{ \frac{ 267000 \ \mathrm{mm^4} \cdot 1 \ \mathrm{} }{ 1050000 \ \mathrm{mm^3} \cdot 430 \ \mathrm{mm} } + \sqrt{ {\left( \frac{ 267000 \ \mathrm{mm^4} \cdot 1 \ \mathrm{} }{ 1050000 \ \mathrm{mm^3} \cdot 430 \ \mathrm{mm} } \right)}^{ 2 \ \mathrm{} } + 6.76 \ \mathrm{} \cdot {\left( \frac{ 0.7 \ \mathrm{} \cdot 345 \ \mathrm{MPa} }{ 200000 \ \mathrm{MPa} } \right)}^{ 2 \ \mathrm{} } } } \\[4pt] &\therefore L_{r} = 6.31 \ \mathrm{m} \end{align} \]

[AISC Eq. F2-6 equivalent]

\[ \rightarrow L_{p} \ < \ L_b \ \leq \ L_{r} \]

\[ \begin{align} M_{ncr} &= {C_b} \cdot \left( {M_{p}} - \frac{ \left( {M_{p}} - \frac{ {0.7} \cdot {F_y} \cdot {S_x} }{ {1e+06} } \right) \cdot {L_b} - {L_{p}} }{ {L_{r}} - {L_{p}} } \right) \\[4pt] &= 1 \ \mathrm{} \cdot \left( 403.6 \ \mathrm{kN*m} - \frac{ \left( 403.6 \ \mathrm{kN*m} - \frac{ 0.7 \ \mathrm{} \cdot 345 \ \mathrm{MPa} \cdot 1050000 \ \mathrm{mm^3} }{ 1000000 \ \mathrm{} } \right) \cdot 6 \ \mathrm{m} - 2.153 \ \mathrm{m} }{ 6.31 \ \mathrm{m} - 2.153 \ \mathrm{m} } \right) \\[4pt] &\therefore M_{ncr} = 264.8 \ \mathrm{kN*m} \end{align} \]

[AISC F2.2(b) equivalent]

\[ \begin{align} M_{nltb} &= \min{\left( {M_{ncr}}, {M_{p}} \right)} = \min{\left( 264.8 \ \mathrm{kN*m}, 403.6 \ \mathrm{kN*m} \right)} \\[4pt] & \therefore M_{nltb} = 264.8 \ \mathrm{kN*m} \end{align} \]

[AISC Eq. F2-2 equivalent]

1.5. Controlling Strength

Design flexural strength of the section

\[ \begin{align} \phi M_n &= {\phi_{b}} \cdot \min{\left( {M_{ny}}, {M_{nltb}} \right)} = 0.9 \ \mathrm{} \cdot \min{\left( 403.6 \ \mathrm{kN*m}, 264.8 \ \mathrm{kN*m} \right)} \\[4pt] & \therefore \phi M_n = 238.3 \ \mathrm{kN*m} \end{align} \]

\[ \begin{align} Check \ M_u \ & \leq \ \phi M_n \\[4pt] \ 40 \ \mathrm{kN*m} \ & \leq \ 238.3 \ \mathrm{kN*m} \\[4pt] &\therefore OK \end{align} \]
